# Two sum problem 

The **Two Sum** problem is a classic algorithm challenge where the goal is to find two numbers in an array that add up to a given target value. The task is to return the indices of the two numbers such that their sum equals the target.

A common efficient solution uses a **hash map** to store previously seen numbers and their indices, allowing the problem to be solved in **O(n) time complexity**.
